# 批量任务创建时增加任务后缀设置的方案

## 需求背景

在 IOTO Task Center 插件的批量创建任务过程中，用户可以为笔记设置前缀，但无法设置后缀。用户希望在当前的设置前缀的 Modal 中，增加一个可以输入后缀的输入框，这里的内容也是可选的。

## 技术方案

### 1. 数据结构变更

**修改文件**：`src/ui/batchTaskModals.ts`

将 `BatchPrefixModal` 重命名为 `BatchNameAffixModal`，并修改返回值类型：

```typescript
export interface BatchNameAffix {
	prefix: string;
	suffix: string;
}

export class BatchNameAffixModal extends Modal {
	private prefixInput: TextComponent | null = null;
	private suffixInput: TextComponent | null = null;
	private resolvePromise: ((value: BatchNameAffix | null) => void) | null =
		null;
	private isResolved = false;

	constructor(app: Modal['app']) {
		super(app);
	}

	openAndGetValue(): Promise<BatchNameAffix | null> {
		return new Promise((resolve) => {
			this.resolvePromise = resolve;
			this.open();
		});
	}

	onOpen(): void {
		this.setTitle(t('modal.batchNameAffix.title'));

		const descriptionEl = this.contentEl.createEl('p', {
			text: t('modal.batchNameAffix.desc'),
		});
		descriptionEl.addClass('ioto-tasks-center__modal-desc');

		// 前缀输入
		const prefixSetting = new Setting(this.contentEl)
			.setName(t('modal.batchNameAffix.prefix'))
			.setDesc(t('modal.batchNameAffix.prefixDesc'));
		prefixSetting.addText((text: TextComponent) => {
			this.prefixInput = text;
			text.setPlaceholder(t('modal.batchNameAffix.prefixPlaceholder'));
			text.inputEl.addClass('ioto-tasks-center__modal-input');
		});

		// 后缀输入
		const suffixSetting = new Setting(this.contentEl)
			.setName(t('modal.batchNameAffix.suffix'))
			.setDesc(t('modal.batchNameAffix.suffixDesc'));
		suffixSetting.addText((text: TextComponent) => {
			this.suffixInput = text;
			text.setPlaceholder(t('modal.batchNameAffix.suffixPlaceholder'));
			text.inputEl.addClass('ioto-tasks-center__modal-input');
		});

		// 聚焦到前缀输入框
		this.prefixInput?.inputEl.focus();

		// 监听回车键
		const handleKeyDown = (event: KeyboardEvent) => {
			if (event.key === 'Enter') {
				event.preventDefault();
				this.confirm();
			}
		};
		this.prefixInput?.inputEl.addEventListener('keydown', handleKeyDown);
		this.suffixInput?.inputEl.addEventListener('keydown', handleKeyDown);

		const actionsEl = this.contentEl.createDiv({
			cls: 'ioto-tasks-center__modal-actions',
		});

		new ButtonComponent(actionsEl)
			.setButtonText(t('modal.cancel'))
			.onClick(() => this.close());

		new ButtonComponent(actionsEl)
			.setButtonText(t('modal.confirm'))
			.setCta()
			.onClick(() => this.confirm());
	}

	onClose(): void {
		this.contentEl.empty();
		this.resolve(null);
	}

	private confirm(): void {
		const prefix = this.prefixInput?.getValue() ?? '';
		const suffix = this.suffixInput?.getValue() ?? '';
		this.resolve({ prefix, suffix });
		this.close();
	}

	private resolve(value: BatchNameAffix | null): void {
		if (this.isResolved) {
			return;
		}
		this.isResolved = true;
		this.resolvePromise?.(value);
	}
}
```

### 2. 添加后缀处理函数

**修改文件**：`src/tasks-center/batch-task-template.ts`

添加 `applySuffix` 函数：

```typescript
export function applySuffix(name: string, suffix: string): string {
	return `${name ?? ''}${suffix ?? ''}`;
}

export function applyAffix(
	name: string,
	prefix: string,
	suffix: string,
): string {
	return `${prefix ?? ''}${name ?? ''}${suffix ?? ''}`;
}
```

### 3. 修改调用方

**修改文件**：`src/views/iotoTasksCenterView.ts`

修改 `triggerBatchCreateFromTemplate` 方法：

```typescript
async triggerBatchCreateFromTemplate(): Promise<void> {
    const projectName = this.selectedProject;
    if (!projectName || !this.projects.some((project) => project.name === projectName)) {
        new Notice(t('view.notice.currentProjectUnavailable'));
        return;
    }

    const batchConfig = this.getBatchTemplateConfig();
    if (!batchConfig.enabled) {
        new Notice(t('notice.batchCreate.disabled'));
        return;
    }
    if (batchConfig.templates.length === 0) {
        new Notice(t('notice.batchCreate.noTemplates'));
        return;
    }

    const template = await new BatchTemplateSelectModal(
        this.app,
        batchConfig.templates,
        projectName,
    ).openAndGetValue();
    if (!template) {
        return;
    }

    // 修改：使用新的 BatchNameAffixModal
    const nameAffix = await new BatchNameAffixModal(this.app).openAndGetValue();
    if (nameAffix === null) {
        return;
    }

    const items = parseBatchList(template.listContent);
    if (items.length === 0) {
        new Notice(t('notice.batchCreate.emptyContent'));
        return;
    }

    // 修改：传递 prefix 和 suffix
    const confirmed = await new BatchCreateConfirmModal(this.app, {
        templateName: template.name,
        prefix: nameAffix.prefix,
        suffix: nameAffix.suffix,
        projectName,
        items,
        levelTaskTypes: template.levelTaskTypes,
    }).openAndConfirm();
    if (!confirmed) {
        return;
    }

    // 修改：传递 suffix
    await this.executeBatchCreate(template, nameAffix.prefix, nameAffix.suffix, items);
}
```

## 实现要点

### 最小化代码改动

1. **重命名而非新建**：将 `BatchPrefixModal` 重命名为 `BatchNameAffixModal`，保持代码结构清晰
2. **扩展而非替换**：在原有前缀功能基础上添加后缀，不改变用户习惯
3. **统一处理**：使用 `applyAffix` 函数统一处理前缀和后缀的应用

### 用户体验设计

1. **界面布局**：前缀和后缀输入框垂直排列，清晰明了
2. **提示信息**：每个输入框都有独立的标签和描述
3. **快捷操作**：支持回车键快速确认
4. **可选设计**：前缀和后缀都是可选的，用户可以只设置其中一个

### 兼容性处理

1. **向后兼容**：如果用户只输入前缀，后缀为空，行为与之前一致
2. **默认值处理**：后缀默认为空字符串，不影响现有功能

## 测试验证

1. **功能测试**：
    - 创建批量任务时只设置前缀
    - 创建批量任务时只设置后缀
    - 创建批量任务时同时设置前缀和后缀
    - 创建批量任务时不设置前缀和后缀

2. **兼容性测试**：
    - 前缀和后缀为空时，任务名称与之前一致
    - 前缀和后缀同时存在时，正确组合

3. **边界情况**：
    - 前缀或后缀包含特殊字符
    - 前缀或后缀为空格
    - 前缀或后缀很长的情况
